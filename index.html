A brief introduction to encryption

The Cesar Cipher is one of the oldest encryption methods known to man. It is named after Julius Cesar, who used it to send encrypted messages to his subordinates.This encryption method takes all the letters of a plaintext and applies a shift. The shift is a number which dictates how far down the alphabet you must jump to find the encrypted letter. For instance, if we have the letter "E" in plaintext and apply a shift of 3 we would end up with a "B". As you can see, we started with the letter "E", skipped three letters down, and ended up at "B". In order to decrypt a word which has been encrypted with this method, all you have to do is apply the same shift to the encrypted letter and skip 3 letters up the alphabet. The Vigenere Cipher is a method of encrypting text, which builds upon the Cesar Cipher method. This encryption method was (wrongfully) attributed to Blaise de Vig√®nere and so acquired its name.

In the Vigenere Cipher each letter of the plaintext is given an individual shift, in contrast to the Cesar Cipher where the same shift is applied to all the letters in the plaintext. The shift of each letter is provided with a key, which are all letters The key is then put "upon" the plaintext.

As you can see, each letter in the key has an index in relation to its position in the alphabet. Notice that the letter "A" is indexed to 0. That means that B = 1, C = 2, and so on.

In order to decrypt a text encrypted with the Vigenere Cipher, all you need is the correct key and then skip down the alphabet as many letters as the index of the corresponding key letter.

If the key is totally random, and the length of the key is the same as the plaintext, this method of encryption is more or less unbreakable.
